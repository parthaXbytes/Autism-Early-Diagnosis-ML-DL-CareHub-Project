
# 🧠 Performance Evaluation of ML and DL Approaches for Early Diagnosis of Autism Spectrum Disorder and Development of Autistic Care Hub

## 📌 Project Overview

This project presents a **comprehensive comparative study of Machine Learning (ML) and Deep Learning (DL) models** for the **early diagnosis of Autism Spectrum Disorder (ASD)** using behavioral and demographic data derived from AQ-10 screening tests. Based on the best-performing model, a **web-based Autism Care Hub** was developed to support early screening, diagnosis assistance, and continuous monitoring.

The work focuses on identifying the most reliable algorithms for ASD prediction while emphasizing the importance of **behavioral features over demographic attributes**.

---

### 🔹 Machine Learning

* Logistic Regression
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest
* XGBoost

### 🔹 Unsupervised Learning (Clustering)

* K-Means
* Agglomerative Clustering
* Spectral Clustering
* Gaussian Mixture Model (GMM)
* BIRCH

### 🔹 Deep Learning

* Dense Neural Network
* Dropout Model
* Batch Normalization Model
* LSTM
* Residual Network
* Sigmoid-based Model

### 🔹 Feature Selection

* Random Forest feature importance
* Strong emphasis on behavioral indicators from AQ-10 screening

## 🌐 Autism Care Hub (Web Application)

A **user-friendly web platform** was developed to enable early ASD screening using a questionnaire-based approach.

### 🔧 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **ML Model:** Support Vector Machine (SVM)
* **Database:** Google Sheets (via API)
* **Deployment:** Netlify (Frontend)

🔗 **Live Website:**
👉 https://autismcares.netlify.app/

> ⚠️ Note: This platform is intended for **preliminary screening only** and does not replace professional medical diagnosis.

## 🎯 Key Contributions

* Extensive comparison of ML, DL, and clustering algorithms on identical ASD datasets
* Identification of **SVM and Logistic Regression** as strong ML classifiers
* Demonstration of **Dense and LSTM models** as top-performing DL architectures
* Development of a **real-world ASD screening web platform**
